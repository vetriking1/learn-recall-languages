The classic gotcha: add to the end is l.append(x) in Python, l.push(x) in TypeScript/Rust, l.Add(x) in C#, and l.add(x) in Java.

Sorting trap: Python's l.sort() orders numbers numerically. TypeScript's l.sort() orders them as strings by default ([1, 10, 2]!) — always pass a comparator (a, b) => a - b.
Recall the four ends: front-add = insert(0,x) / unshift, front-remove = pop(0) / shift, end-add = append / push, end-remove = pop() / pop().
