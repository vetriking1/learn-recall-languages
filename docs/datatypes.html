The core types

Concept Python TypeScript C# Rust C++ Java
Integer x = 42 # int let x = 42; // number int x = 42; let x: i32 = 42; int x = 42; int x = 42;
Float / decimal x = 3.14 # float let x = 3.14; // number (same type!) double x = 3.14; // separate type let x: f64 = 3.14; double x = 3.14; double x = 3.14;
String s = "hello" # str let s = "hello"; // string string s = "hello"; let s: &str = "hello"; // or String std::string s = "hello"; String s = "hello"; // capital S
Boolean b = True # capital T/F let b = true; // lowercase bool b = true; // lowercase let b: bool = true; bool b = true; boolean b = true;
Empty / null x = None let x = null; // or undefined string? x = null; // needs ? let x: Option<i32> = None; // no null int* x = nullptr; String x = null; // objects only

Checking & converting

Operation Python TypeScript C# Rust C++ Java
Check type type(x) typeof x x.GetType() / x is int // compile-time only; use is patterns typeid(x).name() x.getClass() / x instanceof Integer
String → int int("10") Number("10") / parseInt("10") int.Parse("10") "10".parse::<i32>().unwrap() std::stoi("10") Integer.parseInt("10")
Number → string str(10) String(10) 10.ToString() 10.to_string() std::to_string(10) Integer.toString(10)
Numeric cast float(10) 10 as number (implicit) (double)10 10_i32 as f64 // explicit `as` static_cast<double>(10) (double) 10
Key difference: Python splits numbers into int and float; TypeScript has a single number (a 64-bit float).
Watch out: Python has one empty value (None). TypeScript has two: null (set on purpose) and undefined (never assigned).
