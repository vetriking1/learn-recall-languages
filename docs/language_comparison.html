🧭 AllLanguages / Language comparison

🌍Why these six languages?

No single language wins everything. This workspace compares six that cover most real-world trade-offs: scripting, web, enterprise, systems, games, and the JVM ecosystem. Use this page to pick which to learn next — then use topics 01–23 to recall how each one spells it.

Why compare at all? Syntax drifts in memory after you switch stacks. Keeping six runnable references side-by-side is cheaper than re-googling “how do I append to a list in Java again?” every quarter.
