⏳Async / Await ★ concept focus

Non-blocking work — define async functions, await results, run tasks in parallel, and handle errors without freezing the thread.

Not every language spells it the same: Python, TypeScript, and C# have async / await keywords. Java uses CompletableFuture. C++ uses std::async + std::future + .get() (C++20 adds co_await). Rust has async fn but needs a runtime crate (tokio) for real I/O.

Recall the parallel trio: asyncio.gather (Python) · Promise.all (TypeScript) · Task.WhenAll (C#) · CompletableFuture.allOf (Java).
Sequential vs parallel: two await sleep(1) calls one after another take ~2 s; gather / Promise.all on two sleeps take ~1 s total.
