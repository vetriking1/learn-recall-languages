How to run the examples

Python (3.10+):

python python/08_strings.py

TypeScript (one-time setup, then run):

cd typescript && npm install npx ts-node 08_strings.ts

C# (requires dotnet-script):

dotnet tool install -g dotnet-script dotnet script csharp/08_strings.cs

Rust (requires rustc):

rustc rust/08_strings.rs -o rust/08_strings && rust/08_strings

C++ (requires g++):

g++ -std=c++17 cpp/08_strings.cpp -o cpp/08_strings && cpp/08_strings

Java (11+, single-file launch — no compile step):

java java/08_strings.java

Files that read input (06) can be fed via a pipe, e.g. echo "Vetri" | python python/06_io.py
