🔒Encapsulation

Hiding state behind methods that guard the invariants.

Who actually enforces "private"? Python doesn't — __x is just name-mangling by convention. TS enforces private at compile time only. C#, C++, Java and Rust enforce it for real. Rust's boundary is the module, not the class.

Recall: the pattern is the same everywhere — keep the field private, expose a read-only view, and force every change through a method that validates it (here, no withdrawal below zero). The only real difference is how strictly each language stops you reaching in: not at all (Python), at compile time (everyone else), and scoped to the module rather than the type (Rust).
