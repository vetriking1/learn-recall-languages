The classic gotcha: safe lookup with a default is d.get(k, 0) in Python, map.get(k) ?? 0 in TypeScript, dict.GetValueOrDefault(k, 0) in C#, map.get(&k).copied().unwrap_or(0) in Rust, m.count(k) ? m.at(k) : 0 in C++, and map.getOrDefault(k, 0) in Java.
