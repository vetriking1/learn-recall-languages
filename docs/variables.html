Naming convention total_price # snake_case totalPrice // camelCase totalPrice // camelCase (locals) total_price // snake_case (enforced) total_price // snake_case (convention) totalPrice // camelCase
Key difference: Python has no declaration keyword and no real constant (UPPERCASE is just convention). TypeScript uses const by default and let when you need to reassign.
Watch out: Python is dynamically typed — a variable can be rebound to a different type. TypeScript locks the type once inferred (let n = 5; n = "x"; is an error).
