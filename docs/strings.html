The method names differ the most here — this is the recall sheet.

The classic gotcha: length is len(s) in Python, s.length in TypeScript, s.Length (capital L) in C#, s.len() (bytes) or s.chars().count() (Unicode) in Rust, and s.length() (a method) in Java.

Recall: strings are immutable in every language here — each "modifying" method returns a new string. Length is a property in JS/C# but a method in Java (s.length()) and Python (len(s)). Only Python/C#/Rust support negative or from-end indexing.
